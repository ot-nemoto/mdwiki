# Summary

Simple wiki to write in markdown

# Dependencies

- ruby >= 1.9.3-p448
- gem >= 1.8.23
- rails >= 4.0.1

# Getting Started
### Download

    git clone https://github.com/nemonium/mdwiki.git

### Gem package install

    cd mdwiki
    bundle install

### Configure

    cp config/users.yml.example config/users.yml

### Boot

    rails s
